About Metro Raasta

Metro Raasta is an offline route navigator designed for the travelers of Delhi Metro.

Recent features added:

• Responsive GUI
• Autocomplete integeration: Users can now use autocomplete feature while entering source and destination stations
• Nearby Metro Stations: Users can now search metro stations near to their current location
• Metro Map integration: Users can now view full offline map of Delhi Metro
